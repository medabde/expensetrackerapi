# Expense tracker API

An API that helps tracking expenses depending on categories and users

# Features :
 - CRUD Expenses.
 - CRUD Categories.  
 - Login/Register.
 - Security (Login before seeing or making opperations).
 - Password Crypted.
   
# Libraries used :
 - Spring web
 - JDBC API
 - PostgreSQL Driver
 - JWT Authentication
 - BCrypt
